# UIBezierPath
UIBezierPath绘制柱状图、折线图和饼状图

#### 基本原理：
利用UIBezierPath能够创建基于矢量路径的特性来绘制图形的路径，然后将UIBezierPath和CAShapeLayer建立关系，让后者在前者提供的路径中进行渲染，最后生成我们所需的各种图形。而且可以给CAShapeLayer添加动画特效。
